# Jaxley-Models

Jaxley-Models provides a library of common biophysical models implemented using [Jaxley](https://github.com/jaxleyverse/jaxley).

## Installation

Clone the repository:
```bash
git clone https://github.com/jaxleyverse/jaxley-models.git
```

Install the dependencies:
```bash
pip install -e jaxley-models
```

## Models

Currently, the following models are implemented:

- [Pyloric Network](https://scispace.com/pdf/alternative-to-hand-tuning-conductance-based-models-tqk0oa7i15.pdf)
- [CA1]()
- [Layer 5 Pyramidal Cell]()
